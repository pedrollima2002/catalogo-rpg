# Catálogo RPG Multimarcas

Catálogo online desenvolvido para uma loja de moda masculina, permitindo a visualização de produtos, pesquisa, filtros por categoria e contato direto via WhatsApp.

## 🚀 Demonstração

Acesse o projeto online:

https://SEU-USUARIO.github.io/NOME-DO-REPOSITORIO/

## 📋 Funcionalidades

* Exibição dinâmica de produtos a partir de arquivo JSON
* Pesquisa de produtos em tempo real
* Filtros automáticos por categoria
* Página individual para cada produto
* Integração com WhatsApp para contato direto
* Paginação com botão "Carregar Mais"
* Banner personalizado da loja
* Layout responsivo para desktop e dispositivos móveis
* Hospedagem gratuita via GitHub Pages

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* JSON
* Git
* GitHub Pages

## 💡 Objetivo do Projeto

Este projeto foi desenvolvido com o objetivo de criar um catálogo digital para uma loja de roupas, permitindo fácil manutenção dos produtos através de um arquivo JSON e oferecendo uma experiência simples e intuitiva para os clientes.

Além de atender uma necessidade real de negócio, o projeto foi utilizado para aprofundar conhecimentos em JavaScript, manipulação do DOM, consumo de dados externos, organização de código e publicação de aplicações web.
